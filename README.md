# 📋 CBSE Result Portal

A clean and responsive CBSE Class 12 result & percentage calculator built with pure HTML, CSS, and JavaScript — no frameworks, no libraries.

![CBSE Result Portal](https://img.shields.io/badge/HTML-CSS-JS-yellow?style=flat&logo=javascript)

## 🌐 Live Demo

🔗 [hamdan-karadan.github.io/cbse-result-portal](https://hamdan-karadan.github.io/cbse-result-portal)


---

## ✨ Features

- 🎓 Stream selection — PCB, PCM, PCMB
- 📘 English as compulsory subject for all streams
- 💻 Computer Science as an optional subject
- 📊 Subject-wise grade table (A1, A2, B1... E) with individual pass/fail status
- 📈 Overall percentage, grade & final result
- 🖨️ Print / Download result as PDF
- 🌙 Dark mode toggle
- 📱 Mobile responsive design
- ✅ Form validation (roll number, date of birth, age check, marks range)

## 🧮 Grade System

| Percentage | Grade |
|---|---|
| 91 – 100 | A1 |
| 81 – 90 | A2 |
| 71 – 80 | B1 |
| 61 – 70 | B2 |
| 51 – 60 | C1 |
| 41 – 50 | C2 |
| 35 – 40 | D |
| Below 35 | E (Fail) |

> Minimum passing marks per subject: **35 out of 100**
